# eslint-config-flow

## Usage

### eslint-config-flow

install package

```code
npm install --save-dev eslint-config-flow eslint
```

.eslintrc file

```code
{
  "extends": "flow"
}
```

### eslint-config-flow/vue

install package

```code
npm install --save-dev eslint-config-flow eslint eslint-plugin-vue babel-eslint
```

.eslintrc file

```code
{
  "extends": "flow/vue"
}
```
